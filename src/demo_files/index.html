Demo

Basic example:

  <search-select
    ng-model="main.currentFruitOption"
    options="main.fruitOptions"
    option-label-keys="name"
    placeholder-text="Select Fruit"
    font-awesome-icon="fa-lemon-o">
  </search-select>

Results in:

Selected Option: {{main.currentFruitOption}}

Example with multiple label-keys:

  <search-select
    ng-model="main.currentUserOption"
    options="main.userOptions"
    option-label-keys="first_name last_name"
    placeholder-text="Select User"
    font-awesome-icon="fa-user">
  </search-select>

Results in:

Selected Option: {{main.currentUserOption}}
